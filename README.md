Machine Learning HR Employee Attrition Prediction.
Project Overview

The project is aimed at forecasting employee turnover (attrition) in a mid-sized tech company ( TechCorp Solutions ). I created a machine learning pipeline to predict at-risk employees before they quit the company using a dataset of 1,470 employees.

Key Features

Predictive Modeling: Enterprises Logistic Regression and Random Forest.

Data Pipeline: Constructed an entire pipeline with data cleaning, scaling, and feature engineering.

Business Impact: It aims at cutting the voluntary turnover by 20 percent and this may save the company between $ 300,000 to 500,000 in the annual recruitment expenses.

Tools Used

Python (Pandas, Scikit-Learn, Joblib)

Machine Learning (Classification, Pipelines)
